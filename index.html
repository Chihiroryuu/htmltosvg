SVG + SMIL Ctrl+Enter = Apply
💡 SMIL adalah animasi native SVG — animasi tetap berjalan saat file SVG dibuka di browser maupun diembed, tanpa membutuhkan CSS atau JavaScript.

Elemen Animasi Utama

  • <animateTransform> — rotate, scale, translate, skewX/Y
  • <animate> — animasi atribut biasa (fill, opacity, r, dll)
  • <animateMotion> — gerak sepanjang path
  • <set> — langsung set atribut di waktu tertentu

Atribut Penting

  • dur — durasi, misal 3s atau 500ms
  • repeatCount — indefinite atau angka
  • values — list nilai, dipisah titik-koma
  • keyTimes — timing 0–1 tiap value
  • calcMode — spline, linear, discrete, paced
  • keySplines — kurva cubic-bezier per segmen (butuh calcMode="spline")
  • additive="sum" — gabungkan dengan animasi lain pada elemen yang sama
  • begin — delay awal, misal 1s

Rotate vs. Translate (additive)

Untuk kombinasi rotate + translate seperti shackle gembok, gunakan dua <animateTransform> dengan additive="sum" pada elemen yang sama agar keduanya diterapkan sekaligus.

Tips Export

  • SVG — animasi SMIL tetap jalan di browser (Chrome, Firefox, Edge). Tidak jalan di IE.
  • GIF — hasil frame-by-frame, cocok untuk sharing.
  • MP4/WebM — kualitas terbaik, gunakan untuk presentasi.
